# PotatoBot
PotatoBot is a discord utility bot with some features, including:
* triggers on words/phrases with a custom response
* allows users to anonymously send messages to a set channel
  - this is intended for anonymous confessions channel or similar
* some moderation commands
  - ban, tempban, kick, mute
* welcoming new users
  - you can set the welcome message through setting "spawn_message", example "welcome [USER]"\
    [USER] is substituted with the user's username
  - you can set the channel id to send these messages in with setting "spawn_channel"
